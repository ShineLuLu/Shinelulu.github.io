  • Miracl函数库简介
    MIRACL(Multiprecision Integer and RationalArithmetic C/c++ Library)是一套由Shamus Software Ltd.所开发的一套关于大数运算函数库，用来设计与大数运算相关的密码学应用，包含了RSA 公开密码学、Diffie-Hellman密钥交换(Key Exchange)、AES、DSA数字签名，还包含了较新的椭圆曲线密码学(Elliptic CurveCryptography)等等。运算速度快，并提供源代码。MIARCL是当前使用比较广泛的基于公钥加密算法保护实现的大数库之一。

3.Ubuntu下创建C++项目

  • 环境的搭建
    ubuntu操作系统是没有默认带C++编译器g++的，所以需要现安装g++。

按回车即可安装C++的编译器g++。

  • C++工程创建实例

    • 以创建hello工程为例

    • 在ubuntu系统下使用mkdir命令创建一个hello的目录。 在该目录下创建hello.h,hello.cpp,main.cpp,makefile四个文件
      -各文件代码

      • hello.h

      • hello.cpp

      • main.cpp

    • 这三个文件要以空白行结束，否则编译会有警告信息。

      • makefile文件

    • 写好makefile文件后，即可编译工程。在终端中输入make命令，回车后将显示如下信息：
      g++ -c hello.cpp -o hello.o
      g++ -g hello.o main.cpp -I. -o main
      rm -f hello.o
      <<<<<< main is created successfully! >>>>>>
      同样，你也可以不使用makefile文件，而直接在终端上输入以下两行命令,也可以编译这个工程。
